3. Bagaimana visualisasi membantu dalam memahami data?
   Visualisasi sangat membantu dalam memahami data dengan cara:
   -Bar chart memudahkan perbandingan rata-rata nilai antar mata pelajaran secara visual dan cepat
   -Boxplot menunjukkan sebaran data, nilai tengah (median), kuartil, dan outlier dengan jelas
   -Memudahkan identifikasi pola seperti mata pelajaran mana yang nilainya konsisten atau bervariasi
   -Membuat data numerik lebih mudah dipahami dibanding hanya membaca angka dalam tabel
   -Membantu dalam pengambilan keputusan dan presentasi hasil analisis kepada orang lain

   Refleksi Siswa

1. Apa hal baru yang kamu pelajari dari kegiatan analisis dan visualisasi data?
   Hal baru yang saya pelajari adalah:
-Cara menggunakan library Pandas untuk mengolah dan menganalisis data dari file CSV
-Teknik menghitung statistik deskriptif seperti mean, median, modus, max, dan min menggunakan Python
-Cara membuat visualisasi data menggunakan matplotlib dan seaborn
-Pemahaman tentang fungsi groupby() untuk mengelompokkan data berdasarkan kategori tertentu
-Cara menginterpretasikan grafik bar chart dan boxplot untuk memahami pola data
-Pentingnya visualisasi dalam menyampaikan informasi data secara efektif
2. Kesulitan apa yang kamu alami dalam membuat grafik?
   Kesulitan yang dialami antara lain:
-Memahami sintaks dan parameter yang digunakan dalam fungsi plotting (plt.title, plt.xlabel, dll)
-Menyesuaikan ukuran dan tampilan grafik agar lebih mudah dibaca
-Memilih jenis visualisasi yang tepat untuk data yang dianalisis
-Troubleshooting error seperti library yang belum terinstall atau path file CSV yang salah
-Memahami perbedaan antara plot dari matplotlib dan seaborn
-Mengatur label dan judul grafik agar informatif
3. Menurtu kamu AI apa membantu dalam analysis sebua data?
   Ya, AI sangat membantu dalam analisis data karena:
-Otomatisasi: AI dapat mengotomatiskan proses pengolahan data yang berulang dan memakan waktu
-Deteksi Pola: AI mampu mengenali pola dan tren dalam dataset besar yang sulit dilihat secara manual
-Prediksi: AI dapat membuat prediksi berdasarkan data historis (misalnya memprediksi nilai siswa di masa depan)
-Efisiensi: Mempercepat proses analisis data yang kompleks
-Insight Mendalam: AI dapat memberikan wawasan yang lebih dalam melalui machine learning dan analisis lanjutan
-Visualisasi Otomatis: Beberapa tool AI dapat membuat visualisasi data secara otomatis
-Personalisasi: Dalam konteks pendidikan, AI dapat membantu memberikan rekomendasi pembelajaran yang dipersonalisasi berdasarkan analisis nilai siswa
